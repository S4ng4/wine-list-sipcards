Wine JSON Updater

Add new wines to your database by finding the last ID and appending new entries

How this updater works:

  1. Upload new wines JSON: File with wines data (without IDs)
  2. Upload existing wines.json: Your current database from data/ folder
  3. Automatic ID assignment: New wines get IDs starting from last ID + 1
  4. Download updated file: Complete wines.json ready for your project
Expected JSON format for new wines: [ { "producer": "Producer Name", "vintage": "2017", "denomination": "DOC Barbera d'Alba", "grapes": "Barbera 100%", "alcohol_content": "14.5", // ... (no ID field needed) } ]
